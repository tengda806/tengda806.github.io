vscode常用快捷键总结
vscode 快捷键
vscode
发布日期:   2018-06-01
文章字数:   297
阅读时长:   1 分

编辑

Ctrl + X : 剪切选中的内容，如果没有选中内容，剪切光标所在整行内容
Ctrl + C : 复制选中的内容，如果没有选中内容，复制光标所在整行内容
Alt + 下/上 : 移动行
shift + Alt + 上/下 : 复制行
Ctrl+Shift+K : 删除行
Ctrl + / : 行注释
Ctrl + W + F4 : 关闭当前tab
Ctrl + 1/2/3… : 跳到指定tab
Ctrl+Shift+T : 重新打开上一次关闭的tab

查询

Ctrl + F : 搜索
Ctrl + H : 替换
Ctrl + P : 搜索文件
Ctrl + G : 跳转行
Ctrl + Shift + Tab : 切换tab页
Ctrl + Shift + M : 显示终端、错误等程序面板
Alt + 左/右 : 左右切换tab页
Alt + Enter : 选中所有匹配搜索的

文件

Ctrl + N : 新建文件
Ctrl + O : 打开文件
Ctrl + S : 保存
Ctrl+Shift+S : 另存为
Ctrl+K -> S : 保存所有
Ctrl + K -> P : 复制文件路径
Ctrl + K -> R : 在资源管理器中打开文件
Ctrl + K -> O : 在新窗口打开文件

显示

F11 : 全屏
Ctrl+ =/- : 放大或缩小
Ctrl+B : 开关侧边栏
Ctrl+Shift+E : 焦点放到Explorer
Ctrl+Shift+F : 焦点放到搜索,改成ctrl+alt+f
Ctrl+Shift+G : 焦点放Git
Ctrl+Shift+D : 焦点放到Debug
Ctrl+Shift+X : 焦点放到扩展
